日文動詞「する」變化大全

我創建這個網頁的目的，是為了幫助和我一樣曾經為日文動詞變化感到混亂的人。

還記得我第一次看到「する、しない、します、した、される、させられる」這一大串形式時，真的是一個頭兩個大。但隨著學習深入，我發現只要掌握變化的規則和語法邏輯，再搭配反覆練習，其實這些看似複雜的變化都能變得簡單易懂。

這個頁面特別是針對正在學習日文 N5 和 N4 等級的學習者設計，幫助大家建立動詞變化的基本概念與應用能力。如果你和我一樣已經是 N3 或以上 的程度，那你可能已經對這些內容相當熟悉，就不一定需要這樣的整理。不過，作為複習或教學輔助，這個網頁仍然可能對你有幫助。

1. 基本形式

形式變化說明發音
基本形する最基本、不加修飾的動詞形式
ます形します禮貌形
否定形しない不做
過去形した做了
否定過去しなかった沒有做

2. 敬語形式

形式變化說明發音
敬語肯定しました禮貌過去形
敬語否定しません不做（敬語）
敬語否定過去しませんでした沒有做（敬語）

3. 意願・允許・完成

形式變化說明發音
希望したい想做
希望否定したくない不想做
希望（過去）したかった以前想做
希望否定（過去）したくなかった以前不想做
允許してもいい / していい可以做
不必做しなくてもいい / しないでもいい可以不做
完成 / 遺憾してしまう / してしまった做完了（或語氣帶有遺憾）

以下的動詞變化，無論是「基本型」還是「禮貌型」，都可以依照這些基本變化進行。基本型是動詞的最基本形式，禮貌型則是在基本型上加上「ます」語尾來表示禮貌。接下來的變化，例如「たい型」和「ていい型」，也會根據這些基本型和禮貌型的模式進行。這些形式的變化會根據語氣和用法的不同進行調整，但規則大致相同。

4. 使役・被動・使役被動

形式變化說明發音
使役させる讓某人做
使役否定させない不讓做
使役過去させた讓做了
使役否定過去させなかった沒有讓做
被動される被做
被動否定されない沒被做
使役被動させられる被迫做
使役被動否定させられない沒被迫做

5. 命令・禁止・請求

形式變化說明發音
命令形して / してください / しろ / せよ做吧（男性、強勢語氣）


禁止形しないで / しないでください

してはいけない / しちゃいけない / してはならない		請不要做

禁止做


6. 義務・必要性

形式變化說明發音
必須しなければならない / しなくてはいけない / しなくちゃいけない必須做

口語必須しなきゃ / しなきゃいけない口語「得做」
