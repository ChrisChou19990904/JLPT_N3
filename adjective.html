い形容詞 vs な形容詞：差別與用法

什麼是い形容詞與な形容詞？

日語的形容詞分為兩類：い形容詞與な形容詞。它們的接續方式、活用規則與句法用法有所不同。

い形容詞（い‑adjective）

  • 以「〜い」結尾，如：高い、新しい、寒い。
  • 可直接修飾名詞：高い山、新しい本
  • 可單獨構成述語句（辭書型句子）：彼は面白い。
  • 加「です」為禮貌語氣，類似動詞的「ます」：面白いです、高くないです
  • 活用：
    • 否定：〜くない（例：高くない）
    • 過去：〜かった（例：高かった）
    • 過去否定：〜くなかった（例：高くなかった）

例句：

このビルは高いです。／このビルは高くないです。

昨日は寒かったです。／昨日は寒くなかったです。

な形容詞（な‑adjective）

  • 不以「い」結尾，多為漢語形容詞（如：静か、便利）
  • 修飾名詞時需加「な」：静かな町、便利な店
  • 不能單獨構成句子，需加「だ」（辭書型）或「です」（丁寧體）：
    彼は静かだ。／彼は静かです。
  • 辭書型的過去式為「だった」，辭書型的現在否定為「じゃない」，辭書型的過去否定為「じゃなかった」，丁寧體的過去式為「でした」：
    昨日の町は賑やかだった。／昨日の町は賑やかでした。／昨日の町は賑やかじゃなかった。

例句：

この町は静かです。／この町は静かじゃないです。

昨日は便利でした。／昨日は便利じゃなかったです。

綜合比較表

項目 い形容詞 な形容詞
名詞修飾 〜い + 名詞（例：高い山） 〜な + 名詞（例：静かな町）
基本敘述（現在肯定） 〜い / 〜いです（例：面白い / 面白いです） 〜だ / 〜です（例：静かだ / 静かです）
否定形 〜くない（例：高くない） 〜じゃない（例：静かじゃない）
過去形 〜かった（例：高かった） 〜だった（例：静かだった）
過去否定 〜くなかった（例：高くなかった） 〜じゃなかった（例：静かじゃなかった）
副詞形 〜く（例：速く） 〜に（例：静かに）

形容詞的假定型（如果...）

  • たら形：
    • い形容詞 → 去い加上かったら：高かったら（如果高的話）
    • な形容詞 → だったら：静かだったら
  • ば形：
    • い形容詞 → 去い加上ければ：高ければ
    • な形容詞 → であれば：静かであれば

形容詞與「なる」的關係

形容詞變副詞後搭配「なる」可表示「變得怎樣」的意思，這是非常常見且基本的日語表達。

  • い形容詞 → 去い加上「くなる」：寒い → 寒くなる
  • な形容詞 → 加上「になる」：静か → 静かになる

天気が寒くなった。

子どもが静かになりました。

動詞的「たい型」、「ない型」與い形容詞的共通點

動詞的「たい」形、「ない」形在語尾變化上與い形容詞幾乎相同：

  • たい型： 行きたい（I want to go） → 行きたかった（I wanted to go）／行きたくない（I don't want to go）／行きたくなかった（I didn't want to go）
  • ない型： 行かない → 行かなかった（過去否定）
  • 這些變化也能搭配「なる」使用： 行きたくなる（變得想去）／行きたくなくなる（變得不想去）

形容詞的連用型（接續：既...又...）

日語中的形容詞可以使用連用型來連接多個形容詞，表達“既...又...”的意思。這種用法在日語中非常常見，通常用來描述一個人或事物的多重特徵。

い形容詞的連用型

  • 當兩個或更多的い形容詞需要連接時，會將「い」去掉並加上「くて」，形成連用型。這樣的句型可以表示“既...又...”的意思。
  • 例如：
    • 彼女は優しくて美しい。（她既溫柔又美麗。）
    • 彼は背が高くてハンサムだ。（他既高大又英俊。）

な形容詞的連用型

  • 對於な形容詞，連用型是將「な」去掉並加上「で」，同樣可以表示“既...又...”的意思。
  • 例如：
    • 彼はハンサムで優しい。（他既英俊又善良。）
    • この町は静かで美しい。（這個城市既安靜又美麗。）

連用型的注意事項

  • 連用型的結構在句子中常常用來描述一個人或物的多個特質。這是表達事物特徵的一種自然且流暢的方式。
  • 在口語中，這種用法尤為常見，尤其是在描述人、物品、或是場所時。

例句：

彼女は優しくて美しい。

彼は背が高くてハンサムだ。

彼はハンサムで優しい。

進階補充：「〜そう」「〜にくい／やすい」「〜すぎる」

  • 〜そうだ（看起來...）
    - い形容詞去「い」加上「そうだ」：おいしそうだ
    - な形容詞直接加「そうだ」：便利そうだ
  • 〜にくい／〜やすい（難／易）
    - 搭配動詞使用：この鉛筆は書きにくい（翻成英文是"This pencil is hard to write with"。記住，是「書きにくい」，而不是「書くにくい」）、テコンドーは学びやすいと思う（翻成英文是"I think Taekwondo is easy to learn"）
  • 〜すぎる（過於...）
    - い形容詞去い加すぎる：高すぎる
    - な形容詞直接加すぎる：静かすぎる

このケーキはおいしそうです。

この部屋は便利そうです。

この店は高すぎる。

静かに話しやすい場所です。

以上為い形容詞與な形容詞的基礎與進階差異整理。
