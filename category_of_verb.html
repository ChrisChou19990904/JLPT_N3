日文動詞變化大全：一類、二類、三類動詞

本網頁旨在幫助日文初學者理解和掌握日文動詞的變化。日文動詞分為三類：一類動詞、二類動詞、三類動詞，每一類動詞的變化規則不同。本文將介紹各類動詞的基本變化，包括「て型」、「た型」、「命令型」、「意量型」、「可能型」、「ない型」等。

日語動詞變化口訣

一類動詞變化口訣

一類動詞的變化規則有一些固定的口訣，這些規則適用於大部分的動詞。

TE型變化

一類動詞的 TE型（て形）變化規則如下：

  • う段 → って
  • つ段 → って
  • る段 → って
  • く段 → いて
  • ぐ段 → いで
  • す段 → して
  • む、ぶ、ぬ → んで

舉例：

  • ある → あって
  • なる → なって
  • 書く（かく） → 書いて（かいて）
  • 泳ぐ（およぐ） → 泳いで（およいで）
  • 話す（はなす） → 話して（はなして）
  • 言う（いう） → 言って（いって）
  • 待つ（まつ） → 待って（まって）
  • 持つ（もつ） → 持って（もって）
  • 歩く（あるく） → 歩いて（あるいて）
  • 走る（はしる） → 走って（はしって）
  • 進む（すすむ） → 進んで（すすんで）
  • 遊ぶ（あそぶ） → 遊んで（あそんで）
  • 死ぬ（しぬ） → 死んで（しんで）

**注意**：「行く」是例外：

  • 行く（いく） → 行って（いって）

TA型變化（其實和TE型一樣）

一類動詞的 TA型（た形）變化規則如下：

  • う段 → った
  • つ段 → った
  • る段 → った
  • く段 → いた
  • ぐ段 → いだ
  • す段 → した
  • む、ぶ、ぬ → んだ

舉例：

  • ある → あった
  • なる → なった
  • 書く（かく） → 書いた（かいた）
  • 話す（はなす） → 話した（はなした）
  • 歩く（あるく） → 歩いた（あるいた）
  • 進む（すすむ） → 進んだ（すすんだ）
  • 遊ぶ（あそぶ） → 遊んだ（あそんだ）
  • 死ぬ（しぬ） → 死んだ（しんだ）

**注意**：「行く」是例外：

  • 行く（いく） → 行った（いった）

命令型變化

一類動詞的 命令型（命令形）變化規則如下：

  • う段 → え
  • つ段 → て
  • る段 → れ
  • く段 → け
  • ぐ段 → げ
  • す段 → せ
  • む → め
  • ぶ → べ
  • ぬ → ね

舉例：

  • なる → なれ
  • 書く（かく） → 書け（かけ）
  • 待つ（まつ） → 待て（まて）
  • 話す（はなす） → 話せ（はなせ）
  • 歩く（あるく） → 歩け（あるけ）
  • 黙る（だまる） → 黙れ（だまれ）

意量型變化

一類動詞的 意量型變化規則如下：

  • う段 → おう
  • つ段 → とう
  • る段 → よう
  • く段 → こう
  • ぐ段 → ごう
  • す段 → そう
  • む段 → もう
  • ぶ段 → ぼう
  • ぬ段 → のう

舉例：

  • なる → なろう
  • 書く（かく） → 書こう（かこう）
  • 泳ぐ（およぐ） → 泳ごう（およごう）
  • 話す（はなす） → 話そう（はなそう）
  • 言う（いう） → 言おう（いおう）
  • 歩く（あるく） → 歩こう（あるこう）

舉例：

  • 書く（かく） → 書こう（かこう）
    意思： 我們來寫吧。
    解釋： 「書こう」是「書く」的意量型，表示一種提議或者決定要做某事的意思，通常用來表示「想做某事」或「讓我們一起做某事」。例如這裡是提議一起寫某些東西。
  • 話す（はなす） → 話そう（はそう）
    意思： 我們來聊聊吧。
    解釋： 「話そう」是「話す」的意量型，這裡表示說話者的提議或計劃，強調「打算」或者「邀請別人一起做某事」。例如「我們一起來聊聊」的意思。
  • 歩く（あるく） → 歩こう（あるこう）
    意思： 我們一起走走吧。
    解釋： 「歩こう」是「歩く」的意量型，這裡表示「我們一起走」的提議或邀請。這類用法通常表示一種輕鬆、非正式的提議。
  • 泳ぐ（およぐ） → 泳ごう（およごう）
    意思： 我們一起游泳吧。
    解釋： 「泳ごう」是「泳ぐ」的意量型，表示一起進行某個活動或行為。這裡是一種提議，「我們一起去游泳」的語氣。
  • 買う（かう） → 買おう（かおう）
    意思： 我們去買吧。
    解釋： 「買おう」是「買う」的意量型，這是一種計劃或者邀請的表達方式，表示說話者和聽者一起去做某事的打算。
  • 作る（つくる） → 作ろう（つくろう）
    意思： 我們來做吧。
    解釋： 「作ろう」是「作る」的意量型，這裡強調「一起做某事」的提議或決心。例如，可以用來提議共同做一個作品或計劃。
  • 行く（いく） → 行こう（いこう）
    意思： 我們一起去吧。
    解釋： 「行こう」是「行く」的意量型，這裡表示「一起去某地」的提議或計劃，通常用來表示未來的行動。

意量型變化（搭配「している」和「思っている」）

以下是「意量型」搭配「している」和「思っている」來表示試圖做某事或計劃做某事的例子：

搭配「している」表示「試著做某事」：

  • 日本語を話そうとしている。
    意思： 他正在嘗試講日語。
    解釋： 「話そうとしている」是「話す」的意量型「話そう」搭配「している」，表示正在嘗試做某事的過程。
  • 駅まで歩こうとしている。
    意思： 他正在試圖走到車站。
    解釋： 「歩こうとしている」是「歩く」的意量型「歩こう」搭配「している」，表示他正在嘗試走過去，這是他行動中的一個努力。

搭配「思っている」表示「打算做某事」或「決定要做某事」：

  • 来月、東京に行こうと思っている。
    意思： 我打算下個月去東京。
    解釋： 「行こう」是「行く」的意量型，搭配「思っている」表示說話者已經有這個計劃，並且這個計劃在進行中。
  • ウルトラマンジャスティスは地球を壊そうと思っていたのに、人間の優しさを見てからコスモスと一緒に地球を守ることにした。
    意思： 超人力霸王正義本打算毀滅地球，但在看到人類善良的一面後決定和高斯一同守護地球。
    解釋： 「壊そう」是「壊す」的意量型，這裡表示某人計劃去做某事，強調他正在考慮並決定要做這件事。

可能型變化

一類動詞的 可能型變化規則如下：

  • う段 → える
  • つ段 → てる
  • る段 → れる
  • く段 → ける
  • ぐ段 → げる
  • す段 → せる
  • む段 → める
  • ぶ段 → べる
  • ぬ段 → ねる

舉例：

  • なる → なれる
  • 書く（かく） → 書ける（かける）
  • 泳ぐ（およぐ） → 泳げる（およげる）
  • 話す（はなす） → 話せる（はなせる）
  • 言う（いう） → 言える（いえる）
  • 歩く（あるく） → 歩ける（あるける）

ない型變化

一類動詞的 nai型（ない形）變化規則如下：

  • う段 → わ（記住，是「わ」而不是「は」喔） + ない
  • つ段 → た　+ ない
  • る段 → ら　+ ない
  • く段 → か　+ ない
  • ぐ段 → が　+ ない
  • す段 → さ　+ ない
  • む段 → ま　+ ない
  • ぶ段 → ば　+ ない
  • ぬ段 → な　+ ない

舉例：

  • なる → ならない
  • 思う（おもう） → 思わない（おもわない）
  • 言う（いう） → 言わない（いわない）
  • 書く（かく） → 書かない（かかない）
  • 話す（はなす） → 話さない（はなさない）
  • 歩く（あるく） → 歩かない（あるかない）
  • 裏切る（うらぎる） → 裏切らない（うらぎらない）
  • 死ぬ（しぬ） → 死なない（しなない）
  • 遊ぶ（あそぶ） → 遊ばない（あそばない）
  • 飛ぶ（とぶ） → 飛ばない（とばない）
  • 読む（よむ） → 読まない（よまない）

**注意**：「ある」是例外：

  • 「ある」直接變成「ない」就好，而不是「あらない」

二類動詞變化口訣

TE型變化

二類動詞的 TE型（て形）變化非常簡單，所有的二類動詞都只需要加「て」。

  • 慣れる（なれる） → 慣れて（なれて）
  • 食べる（たべる） → 食べて（たべて）
  • 覚える（おぼえる） → 覚えて（おぼえて）
  • 褒める（ほめる） → 褒めて（ほめて）
  • 変える（かえる） → 変えて（かえて）
  • 支える（ささえる） → 支えて（ささえて）
  • 吠える（ほえる） → 吠えて（ほえて）
  • 助ける（たすける） → 助けて（たすけて）
  • 続ける（つづける） → 続けて（つづけて）
  • 捨てる（すてる） → 捨てて（すてて）

**注意**：儘管大多數二類動詞結尾為「る」且發音為「ㄟ」，仍有些例外。例如：

  • 生きる（いきる） → 生きて（いきて）
  • 起きる（おきる） → 起きて（おきて）
  • 落ちる（おちる） → 落ちて（おちて）
  • 降りる（おりる） → 降りて（おりて）
  • 借りる（かりる） → 借りて（かりて）
  • 寝る（ねる） → 寝て（ねて）
  • 見る（みる） → 見て（みて）

TA型變化

二類動詞的 TA型（た形）變化同樣非常簡單：

  • 慣れる（なれる） → 慣れた（なれた）
  • 食べる（たべる） → 食べた（たべた）
  • 覚える（おぼえる） → 覚えた（おぼえた）
  • 支える（ささえる） → 支えた（ささえた）
  • 吠える（ほえる） → 吠えた（ほえた）
  • 助ける（たすける） → 助けた（たすけた）
  • 続ける（つづける） → 続けた（つづけた）
  • 捨てる（すてる） → 捨てた（すてた）
  • 寝る（ねる） → 寝た（ねた）
  • 見る（みる） → 見た（みた）
  • 起きる（おきる） → 起きた（おきた）
  • 落ちる（おちる） → 落ちた（おちた）
  • 降りる（おりる） → 降りた（おりた）
  • 生きる（いきる） → 生きた（いきた）

命令型變化

二類動詞的 命令型（命令形）變化直接加「ろ」即可：

  • 慣れる（なれる） → 慣れろ（なれろ）
  • 食べる（たべる） → 食べろ（たべろ）
  • 吠える（ほえる） → 吠えろ（ほえろ）
  • 捨てる（すてる） → 捨てろ（すてろ）
  • 見る（みる） → 見ろ（みろ）
  • 寝る（ねる） → 寝ろ（ねろ）
  • 降りる（おりる） → 降りろ（おりろ）

意量型變化

二類動詞的 意量型（意向形）變化規則如下：

  • 慣れる（なれる） → 慣れよう（なれよう）
  • 食べる（たべる） → 食べよう（たべよう）
  • 覚える（おぼえる） → 覚えよう（おぼえよう）
  • 支える（ささえる） → 支えよう（ささえよう）
  • 吠える（ほえる） → 吠えよう（ほえよう）
  • 助ける（たすける） → 助けよう（たすけよう）
  • 続ける（つづける） → 続けよう（つづけよう）
  • 捨てる（すてる） → 捨てよう（すてよう）
  • 見る（みる） → 見よう（みよう）
  • 起きる（おきる） → 起きよう（おきよう）
  • 落ちる（おちる） → 落ちよう（おちよう）
  • 降りる（おりる） → 降りよう（おりよう）
  • 生きる（いきる） → 生きよう（いきよう）

舉例：

  • 食べる（たべる） → 食べよう（たべよう）
    意思： 我們一起吃吧。
    解釋： 「食べよう」是「食べる」的意量型，表示提議一起做某事，通常用來表達「想要做某事」或「讓我們一起做某事」的意思。
  • 見る（みる） → 見よう（みよう）
    意思： 我們一起看電影吧。
    解釋： 「見よう」是「見る」的意量型，這裡表示說話者提議一起做某事，也有「打算做某事」的含義。
  • 起きる（おきる） → 起きよう（おきよう）
    意思： 讓我們起來吧。
    解釋： 「起きよう」表示提議或計劃一起做某事，這裡用來表示一個「想要做某事」的動作。

可能型變化

二類動詞的 可能型（可能形）變化直接加「られる」即可：

  • 慣れる（なれる） → 慣れられる（なれられる）

舉例：

三類動詞變化口訣

する的變化口訣

「する」是三類動詞中最常見的一個，不規則變化如下：

  • する → して
  • する → した
  • する → しろ / せよ
  • する → しよう
  • する → できる
  • する → しない

来る的變化口訣

「来る」也是三類動詞中的不規則動詞，變化如下：

  • 来る（くる） → 来て（きて）
  • 来る（くる） → 来た（きた）
  • 来る（くる） → 来い（こい）
  • 来る（くる） → 来よう（こよう）
  • 来る（くる） → 来られる（こられる）
  • 来る（くる） → 来ない（こない）

進階變化：被動形・使役形・使役被動形

以下為一類、二類、三類動詞的進階變化整理：

補充說明——

可能形 vs 受身形：一樣的形，不一樣的意思！

對於二類動詞，可能形（表示「能做某事」） 和 受身形（表示「被做某事」） 都使用同樣的變化：〜られる。 判斷它們的方式主要靠上下文語意。

動詞原形 可能形（正式） 可能形（口語・ら抜き言葉） 受身形
食べる 食べられる 食べれる ✅ 食べられる
見る 見られる 見れる ✅ 見られる
起きる 起きられる 起きれる ✅ 起きられる

💡 小技巧：
在口語中，「可能形」常常會省略「ら」，變成「ら抜き言葉」（如：食べれる、見れる）。
這樣的變化只會出現在可能形，不會出現在受身形，所以可以用來幫助區分！

✔️ 例句（可能）：「この寿司、食べれるよ！」
❌ 不能說「*先生に食べれる」來表示「被吃」，這是錯誤的用法。

📝 在正式寫作、考試中請使用完整的「られる」形式。

三類動詞（不規則動詞）

原形 被動形 使役形 使役被動形
する される させる させられる
来る（くる） 来られる（こられる） 来させる（こさせる） 来させられる（こさせられる）

說明

  • 被動形： 表示受到他人動作的影響，例如：地球はウルトラマンに保護されている（地球被超人力霸王保護著）。
  • 使役形： 表示讓他人做某事，例如：ウルトラマンコスモスはカオスヘッダーに怪獣をコントロールさせない（超人力霸王高斯不會讓混沌病毒控制怪獸）。
  • 使役被動形： 表示被迫做某事，例如：ウルトラマンメビウスはウルトラの父に地球へ来させられた（超人力霸王梅比斯被超人力霸王之父要求來地球）。

使役被動形的兩種形式說明

  • 一類動詞的使役被動形有兩種說法：
  • 正式型： 書かせられる、行かせられる（書面語、教科書用）
  • 口語簡略型： 書かされる、行かされる（日常會話常用）

兩者意思相同，可根據情境選擇使用。

進階變化：條件形（ば形・たら形）

表示「如果～的話」的條件語氣，分為書面語（ば形）與口語（たら形）。
