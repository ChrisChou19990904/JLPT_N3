日文動詞變化大全：一類、二類、三類動詞

本網頁旨在幫助日文初學者理解和掌握日文動詞的變化。日文動詞分為三類：一類動詞、二類動詞、三類動詞，每一類動詞的變化規則不同。本文將介紹各類動詞的基本變化，包括「て型」「た型」「命令型」「意量型」等。

1. 一類動詞

形式變化說明發音

て形書いて (kaite)用於連接或請求
た形書いた (kaita)過去式
命令形書け (kake)命令式
意量形書こう (kakou)意圖做某事
基本形行く (iku)「行く」的基本形

た形行った (itta)過去式
命令形行け (ike)命令式

日語動詞變化口訣

一類動詞變化口訣

一類動詞的變化規則有一些固定的口訣，這些規則適用於大部分的動詞。

TE型變化

一類動詞的 TE型（て形）變化規則如下：

  • う段 → って
  • つ段 → って
  • る段 → って
  • く段 → いて
  • ぐ段 → いで
  • す段 → して
  • む、ぶ、ぬ → んで

舉例：

  • 書く（かく） → 書いて（かいて）
  • 泳ぐ（およぐ） → 泳いで（およいで）
  • 話す（はなす） → 話して（はなして）
  • 言う（いう） → 言って（いって）
  • 歩く（あるく） → 歩いて（あるいて）

**注意**：「行く」是例外：

  • 行く（いく） → 行って（いって）

TA型變化

一類動詞的 TA型（た形）變化規則如下：

  • う段 → った
  • つ段 → った
  • る段 → った
  • く段 → いた
  • ぐ段 → いだ
  • す段 → した
  • む、ぶ、ぬ → んだ

舉例：

  • 書く（かく） → 書いた（かいた）
  • 話す（はなす） → 話した（はなした）
  • 歩く（あるく） → 歩いた（あるいた）

**注意**：「行く」是例外：

  • 行く（いく） → 行った（いった）

命令型變化

一類動詞的 命令型（命令形）變化規則如下：

  • う段 → え
  • つ段 → て
  • る段 → れ
  • く段 → け
  • ぐ段 → げ
  • す段 → せ
  • む、ぶ、ぬ → んで

舉例：

  • 書く（かく） → 書け（かけ）
  • 話す（はなす） → 話せ（はなせ）
  • 歩く（あるく） → 歩け（あるけ）
  • 黙る（だまる） → 黙れ（だまれ）

二類動詞變化口訣

TE型變化

二類動詞的 TE型（て形）變化非常簡單，幾乎所有的二類動詞都只需要加「て」。

  • 食べる（たべる） → 食べて（たべて）
  • 覚える（おぼえる） → 覚えて（おぼえて）
  • 支える（ささえる） → 支えて（ささえて）
  • 吠える（ほえる） → 吠えて（ほえて）
  • 助ける（たすける） → 助けて（たすけて）
  • 続ける（つづける） → 続けて（つづけて）

**注意**：儘管大多數二類動詞結尾為「る」且發音為「ㄟ」，仍有些例外。例如：

  • 生きる（いきる） → 生きて（いきて）
  • 起きる（おきる） → 起きて（おきて）
  • 落ちる（おちる） → 落ちて（おちて）
  • 降りる（おりる） → 降りて（おりて）
  • 寝る（ねる） → 寝て（ねて）
  • 見る（みる） → 見て（みて）

TA型變化

二類動詞的 TA型（た形）變化同樣非常簡單：

  • 食べる（たべる） → 食べた（たべた）
  • 寝る（ねる） → 寝た（ねた）

命令型變化

二類動詞的 命令型（命令形）變化直接加「ろ」即可：

  • 食べる（たべる） → 食べろ（たべろ）
  • 吠える（ほえる） → 吠えろ（ほえろ）
  • 寝る（ねる） → 寝ろ（ねろ）
  • 降りる（おりる） → 降りろ（おりろ）

三類動詞變化口訣

する的變化口訣

「する」是三類動詞中最常見的一個，不規則變化如下：

  • する → して
  • する → した
  • する → しろ
  • する → しよう

来る的變化口訣

「来る」也是三類動詞中的不規則動詞，變化如下：

  • 来る（くる） → 来て（きて）
  • 来る（くる） → 来た（きた）
  • 来る（くる） → 来い（こい）
  • 来る（くる） → 来よう（こよう）
